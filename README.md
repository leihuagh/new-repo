# Connect Local Existing Project to GitHub

## Steps

## Working on GitHub:

- Create a Repository

## Working on Local

- First time push

```
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/leihuagh/new-repo.git
git push --set-upstream origin master
```

- Later pushing using

```
git push -u origin master
```
